Interaction states

Button state matrix

Static specimens for implementation and Figma mapping. The same button geometry is preserved across default, hover, focus, active, disabled, and loading states.

Default primary

Base primary action for the one main command on a view.

Hover primary

Focused primary

Active primary

Disabled primary

Disabled state lowers emphasis and removes action affordance.

Loading primary

Loading keeps button width stable and pairs with busy state.
